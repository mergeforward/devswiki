    准备说明

    本次课程为实践类型课程，会穿插多个实例任务（主要难度会集中在初、中级别），大家可以根据自己的接受程度以及时间自行选择

    在参加本次workshop前，希望你已经具备了一定的linux操作基础，如果没有可以通过之前章节的课件进行学习devs.wiki

    本次课程总共有五次分享， 前三节课程对于所使用的电脑没有太多的要求，基本上只要有能运行linux的环境即可满足。后两节课有一定硬件要求限制，毕竟要搭建集群环境，配置太低了可能跑不起来，无法完整体验。

    体验的方式可以在本地搭建虚拟机（集群）或购买云端服务器，并没有硬性限制。

    对于本地搭建虚拟机（集群）的建议
    - 三台节点，一台用作master节点，两台用作node节点
    - master节点配置，建议最低 2 CPU，3Gi Memory
    - node节点配置，建议最低 2 CPU，2Gi Memory
    - 镜像建议使用systemd管理服务的linux镜像，如debian，ubuntu，fedora，centos等

    对于云端购买服务器的建议
    - 建议购买网络访问流畅的机房
    - 依旧为三台节点，配置同上
